# Repository Copilot Instructions

This is a controlled maintenance repository. Do not behave like a general-purpose wiki editor.

Allowed actions:

1. `add_module`
2. `source_sync`

Every change must contain time and source. If source or time is missing, ask for it or create a draft with explicit TODO markers. Never invent provenance.

Before answering or editing, read:

- `CONTROLLED_MAINTENANCE.md`
- `AGENTS.md`
- `MODULE_SYSTEM.md`
- `SOURCE_SYNC.md`

When creating content, preserve front matter and source metadata.

Never process customer data, production secrets, credentials, account numbers, regulated records, or raw production logs.
